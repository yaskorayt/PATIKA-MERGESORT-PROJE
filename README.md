# Merge-Sort
[16,21,11,8,12,22] -> Merge Sort
## Aşamaları
Listeyi her adımda parçalayıp tek eleman kalıncaya kadar böleceğiz. Böldükten sonra sıralı bir şekilde dizmeye başlayacağız.

     [16,21,11,8,12,22]  
*    [16,21,11]  [8,12,22]   
*  [16,21] [11] [8,12] [22]  
* [16] [21] [11] [8] [12] [22]
*  [16,21] [11] [8,12] [22]
*    [11,16,21] [8,12,22]
*   [8,11,12,16,21,22]
    
## Big-O Gösterimi
O(n²)’ dir.
